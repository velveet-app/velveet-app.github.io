Built to last

Solid stack,
no cloud dependency

Velveet runs entirely on your machine. No account, no telemetry, no subscription. A local SQLite database, a lightweight Node.js server, and a React front-end served at localhost.

React TypeScript Node.js Prisma / SQLite FFmpeg Redux Toolkit SCSS Design System
lock 100% local

All data stays on your machine. Nothing is sent anywhere, ever.

speed Fast & responsive

Local network speeds. No buffering, no CDN latency, no throttling.

source Open source

MIT licensed. Inspect, fork, or contribute on GitHub.
